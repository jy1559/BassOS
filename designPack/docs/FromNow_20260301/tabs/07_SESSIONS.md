# 세션 기록 탭 상세

## 역할
- 실제 연습 세션 로그를 표 형태로 조회/수정/삭제
- 짧은 통계와 활동별 분포를 빠르게 확인

## 필터
- 기간: `7d`, `30d`, `all`
- 활동: `Song`, `Drill`, `Etc`
- 검색: 곡/드릴/노트/태그

## 상단 요약
- 세션 수
- 총 시간(분)
- 평균 세션 시간
- 총 XP
- 노래 세션 수
- 드릴 세션 수

## 활동 분포
- 활동별 누적 시간 bar

## 표 컬럼
- 시작(날짜+시간 줄바꿈)
- 활동
- 세부
- 곡/드릴(있는 값만 표시)
- 속도/BPM
- 분
- XP
- 노트
- 관리(수정/삭제)

## 수정 가능한 항목
- 시작/종료 시간
- 활동/세부활동
- 태그
- 노트

## 삭제 동작
- 세션 삭제 시 XP 및 레벨 재계산(역반영)

## 관련 API
- `GET /api/sessions`
- `PUT /api/sessions/{event_id}`
- `DELETE /api/sessions/{event_id}`

## 현재 제약/주의
- 기간 필터가 고정 3단계라 상세 기간 탐색은 돌아보기 탭이 담당
- 복합 정렬/그룹화 UI는 아직 없음
