# 마이 XP 탭 상세

## 역할
- 레벨/랭크/배지 상태와 XP 획득 구조를 보여주는 개인 성장 탭

## 기간 필터
- `7d`, `30d`, `all`
- 아래 카드(XP source/activity/timeline)에 공통 적용

## 카드 구성
1. 현재 XP / 레벨
- Total XP, Level, Rank, 다음 레벨까지 XP
- 배지 표시, 레벨 타이틀, 응원 문구

2. XP 획득 비중
- 출처별 파이 + 막대
- 출처 키 예: `practice`, `quest`, `achievement`, `long_goal`, `other`

3. 기간별 XP 추이
- 기간 단위 미니 차트

4. 활동별 XP
- activity 기준 XP 분포

5. 레벨 그래프 / 다음 해금
- 최근 레벨 타임라인
- upcoming unlock 3개

## 데이터 소스
- `GET /api/player/xp`
  - `hud`, `stats`, `xp_sources`, `xp_by_activity`, `xp_timeline`, `upcoming_unlocks`

## 현재 제약/주의
- “현재 XP/레벨” 카드에는 기간 필터 영향이 없어 혼동 여지 있음
- 활동명 표준화(영문 키/한글 라벨 정합성)는 추가 정리 필요
