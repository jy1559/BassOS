# 돌아보기 탭 상세

## 역할
- 세션 기록을 기간별로 압축해서 성취감 중심으로 보여주는 분석 탭
- “연습량 정리 + 포트폴리오 상태”를 한 화면에서 확인

## 기간 선택
- 모드: `주별`, `월별`, `연별`, `선택(custom)`
- 비-custom 모드: 좌/우 화살표로 기간 이동
- custom 모드: 시작/종료 날짜 직접 선택

## 주요 카드
1. 누적 성과
- 세션 수, 연습 시간, XP, 연습한 곡/드릴 수
- 현재/최장 streak
- 전체 누적 세션 수

2. 곡 포트폴리오
- 시작 전/진행 중/완료 분포

3. 곡 목적별 연습 시간
- purpose 기준 시간 분포

4. 드릴 영역별 연습 시간
- area 기준 시간 분포

5. TOP 곡/드릴
- 분/세션수 기준 상위

6. 월별 누적 XP
- 월 단위 XP bar

## 데이터 소스
- 세션: `events.csv`
- 곡 상태/목적: `song_library.csv`
- 드릴 영역: `drill_library.csv`

## 현재 제약/주의
- 활동별 세부 분류(예: sub_activity, 태그 조합)는 아직 별도 카드가 적음
- 파이/히트맵 등 시각화 종류는 향후 확대 가능
