# 연습 도구 탭 상세

## 역할
- 메트로놈 + TAB 메이커를 한 탭에서 제공
- 연습 실행 중 보조 도구로 사용

## 메트로놈(글로벌 도크)
- 상단 우측 도크에서 열고 닫기
- 탭 이동해도 재생 상태 유지(Provider 전역 상태)
- 기능
  - BPM 입력/적용 + 슬라이더(30~260)
  - 박자(분자/분모)
  - 강박 위치 입력(예: `1,3`)
  - 볼륨
  - 비주얼 비트 라이트(강박 강조)
  - 고급 설정(Hz, wave, gain)

## TAB 메이커
- 최소 4마디 단위(4의 배수)
- 노트 속성
  - stringNo(1~4)
  - fret
  - rest 여부
  - 길이: `1/16`, `1/8`, `1/4`, `1/2`, `1`
  - modifier: `normal`, `dot`, `triplet`
- 단축키
  - `Q/W/E/R/T`: 길이
  - `A`: 쉼표 토글
  - `S`: 점음표 토글
  - `D`: 셋잇단 토글
  - `F`: modifier 초기화
  - `Ctrl+Z`: 실행취소
  - `Ctrl+C/V`: 마디 복사/붙여넣기
  - 화살표: 줄 이동
  - 숫자키: 프렛 입력
- 기능
  - JSON 저장/불러오기
  - 이미지 내보내기
  - 재생(cursor 기반)

## 관련 상태
- 메트로놈: `metronome.tsx` Context
- TAB 프로젝트: `TabProject` (런타임 상태)

## 현재 제약/주의
- TAB 입력은 강력하지만 초보자 온보딩이 부족
- 악보 시각화가 텍스트 기반이라 “실제 탭지” 느낌은 개선 여지 큼
- 재생은 베이직 구현이며 음색/표현 확장 여지 큼
