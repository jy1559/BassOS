# BASSOS 프로그램 개요 (2026-03-01 기준)

## 1) 한 줄 정의
BASSOS는 **베이스 기타 연습을 로컬에서 게임처럼 기록/시각화**하는 Windows 전용 데스크톱 앱입니다.

## 2) 핵심 목표
- 기록 진입장벽 최소화: 시작/종료 중심의 빠른 세션 기록 UX
- 몰입 유지: 레벨/배지/해금/업적/퀘스트로 성취감 제공
- 로컬 안정성: CSV+JSON 기반 저장, 오프라인 친화, exe 배포
- 가시화: 내 연습 흐름을 세션/주간/월간/누적으로 명확히 확인

## 3) 제품 성격
- 대상: 개인 베이스 연습자(단일 사용자)
- 환경: Windows
- 배포: `PyInstaller` 기반 exe
- 실행 구조: `Flask API + React(Vite) UI + PyWebView`

## 4) 현재 기능 축(대분류)
- 대시보드: HUD, 타이머, 진행률, TODO/퀘스트, 곡 바로가기, 이미지 카드
- 연습: 연습 스튜디오(곡/드릴/배킹), 드릴 리스트, 연습 도구(메트로놈/탭)
- 기록: 세션 기록, 돌아보기, 마이 XP, 기록장(게시글+첨부)
- 도전: 퀘스트, 업적
- 시스템: 설정, 내보내기/백업, 빌드

## 5) 유지해야 할 제품 원칙
- “보기 좋은 것”보다 “연습이 계속되게 만드는 것” 우선
- 데이터는 설명 가능해야 함(왜 XP가 올랐는지 추적 가능)
- 탭/카드 수가 늘어도 검색/필터/그룹화 우선으로 스케일 대응
- 화면이 작아져도 핵심 HUD/타이머/기록 입력이 깨지지 않게 반응형 유지
