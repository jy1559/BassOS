# 개선 백로그 (우선순위)

## P0 (바로 손대는 영역)
1. 대시보드 반응형 안정화
- 목표: 어떤 창 크기에서도 스크롤 없이 핵심 카드(HUD/타이머/진행)가 보이게
- 완료 기준
  - 최소 창에서 HUD 수치 줄바꿈 깨짐 없음
  - 곡 바로가기 썸네일이 정사각 유지

2. 연습 스튜디오 레이아웃 단순화
- 목표: 곡/드릴 모드별 핵심 컨텐츠 우선 노출
- 완료 기준
  - 곡 모드 iframe이 주 영역
  - 드릴 모드 이미지+배킹 분할 균형

3. 기록장 작성 패널 정리
- 목표: 본문 중심 작성 UX
- 완료 기준
  - 본문 입력 고정 큰 영역
  - 태그/연결/첨부는 접힘 또는 탭 분리

4. 테스트 관련 모의 데이터 추가
- 목표: 테스트 할 때 볼 모의 데이터 추가
- 완료 기준
  - 세팅의 critical setting 부분에 "모의 데이터 추가" 및 제거 버튼 추가
  - 모의데이터 추가 시 곡 라이브러리에 곡 추가 및 세션 진행 등 2달동안 사용한 모의 데이터 추가
  - 곡 라이브러리에 임의로 곡 (추천 곡중에 몇 개 골라서) 라이브러리 추가
  - 모의 데이터는 열심히 한 주 + 1주 이상 자리 비운 적도 있음 + 몰아서 공연연습 등 사실적으로 모의 데이터 추가해서 돌아보기 탭 등 테스트 가능하게

## P1 (밸런스/동기부여)
1. XP 밸런스 시뮬레이터
- settings에서 파라미터 변경 시 성장 예상치 즉시 계산

2. 배지/칭호 체계 고도화
- 티어 시작 단순/티어 말기 화려 패턴 적용
- 다음 레벨 배지 미리보기

3. 업적 정리
- 업적 설명 툴팁 정리
- 태그 한글화와 데이터 정합성 점검

## P2 (중기 확장)
1. 대시보드 위젯 템플릿 프리셋
- compact/focus/analysis 모드

2. 기록장 고급 편집
- 본문 중간 첨부 배치
- 블록형 에디터(텍스트/미디어 섹션)

3. 분석 고도화
- 돌아보기에 heatmap/코호트/장르별 추세 추가

## 추천 진행 순서
1. P0-1 → P0-2 → P0-3
2. P1-1 → P1-2 → P1-3
3. P2 병행
